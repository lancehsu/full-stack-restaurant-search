# Restaurant search web app example

## To Use

### Clone this repo

```zsh
git clone git@github.com:lancehsu/glints-assessment.git
```

### Build Front-end and run server

> `PORT: 8000`

1. Build front-end and run server

```zsh
yarn install
yarn local-start
```

2. Go http://localhost:8000/
